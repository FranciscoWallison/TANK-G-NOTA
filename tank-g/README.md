# M-VAVE TANK-G — Documentação Completa (PT-BR)

Pedal multiefeitos de guitarra portátil com **36 presets editáveis**, bateria interna, Bluetooth, USB sound card e saída XLR balanceada.

> Este material é uma reorganização didática do manual oficial M-VAVE para uso pessoal. Os nomes de marcas/modelos de amps simulados são marcas registradas dos respectivos proprietários e não têm relação com a M-VAVE — são citados apenas para indicar o caráter sonoro.

## ⚡ Início rápido (3 minutos)

1. **Ligue** o TANK-G no botão de power (parte de trás)
2. **Conecte a guitarra** no INPUT (1/4") e um fone na saída PHONES
3. O pedal inicia em **modo PRESET** mostrando o BANK no display
4. Use **BANK-/BANK+** + **A/B/C/D** para navegar pelos 36 presets
5. Para editar em tempo real: pise em **B + C juntos** → entra no **modo LIVE**

> Detalhe completo em [08-guia-pratico.md](08-guia-pratico.md).

## 🎛️ Atalhos essenciais (decora)

| Combo | Ação |
|-------|------|
| **B + C** (tap) | Alterna entre modo **PRESET ↔ LIVE** |
| **B + C** (segurar 2s) | Liga/desliga o **AFINADOR (TUNER)** |
| **B + C** (segurar 3s) | Liga/desliga o **BLUETOOTH (BT)** |
| **BANK-** / **BANK+** | Diminui / aumenta o banco (1–9) |
| **A / B / C / D** | Seleciona preset dentro do banco |
| Segurar **A/B/C/D** no modo LIVE | **Salva** edição naquele slot |
