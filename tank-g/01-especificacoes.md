# 01 — Especificações Técnicas

## 🎚️ Recursos principais

| Recurso | Detalhe |
|---------|---------|
| Presets editáveis | **36** (9 bancos × 4 slots A/B/C/D) |
| Bateria interna | Sim — autonomia **~12 horas** |
| Carregamento | USB-C, **5V / 2A**, **~4h** para carga completa |
| Noise Gate | Sim (com botão de threshold) |
| Pré-amplificadores | **9 tipos** simulados |
| EQ | 3 bandas (Treble / Middle / Bass) |
| Modulações | **3 tipos** (Chorus / Phaser / Tremolo) |
| Delays | **3 tipos** (Analog / Tape / Dual) |
| Reverbs | **3 tipos** (Room / Spring / Cloud) |
| IR CAB | **8 slots** + 1 slot OFF (total 9 posições) — importáveis |
| Afinador (Tuner) | Sim, cromático com indicação de # |
| Bluetooth | Sim (para áudio / controle via app) |
| Saídas | OUTPUT A 1/4", OUTPUT B 1/4", PHONES, XLR balanceada |
| Entrada | INPUT 1/4" mono |
| USB | Carregamento + sound card de PC + edição de presets + import de IR |
| Gravação celular | Sim (via cabo de gravação específico) |

## 🔋 Alimentação

- **Não use fonte de pedal tradicional (9V).** O TANK-G é alimentado por **USB-C 5V**.
- Carregador recomendado: **5V / 2A** (carregador comum de celular serve, desde que entregue 2A).
- Power bank funciona — útil para shows sem tomada.
- **Indicador de bateria fraca:** o display pisca a letra **"L"** a cada 5 segundos. Quando isso acontecer, **carregue o quanto antes** — operação prolongada com bateria fraca pode causar instabilidade no som.
- **Indicador LED de carga:**
  - 🔴 Vermelho = carregando
  - 🔵 Azul = carga completa
  - Apagado = uso normal em bateria

## 🌡️ Cuidados / Onde NÃO colocar

Evite expor o pedal a:

- Fontes de calor
- Campos magnéticos fortes
- Luz solar direta
- Locais com alta temperatura ou umidade
- Locais empoeirados ou sujos
- Vibrações ou choques fortes

## ⚠️ Conexão

Antes de conectar/desconectar cabos, **desligue o equipamento associado** (ampli, mixer, interface) para evitar pops e danos aos alto-falantes.

## 📡 Interferência

Ondas eletromagnéticas de rádio, TV e outros equipamentos podem interferir no áudio. **Mantenha o TANK-G afastado** desses dispositivos.
