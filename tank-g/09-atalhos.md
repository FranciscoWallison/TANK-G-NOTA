# 09 — Atalhos / Cheat Sheet

Imprima e cola em cima do pedal. 😄

---

## 🦶 Footswitches

### Modo **PRESET** (padrão ao ligar)

| Footswitch | Ação |
|------------|------|
| **A** | Carrega preset A do banco atual |
| **B** | Carrega preset B |
| **C** | Carrega preset C |
| **D** | Carrega preset D |
| **BANK-** | Diminui banco (display pisca, aguarda A/B/C/D para confirmar) |
| **BANK+** | Aumenta banco |

### Modo **LIVE** (edição em tempo real)

| Footswitch | Liga/desliga módulo |
|------------|---------------------|
| **A** | **REVERB** |
| **B** | **DELAY** |
| **C** | **MOD** |
| **D** | **AMP** |

---

## ⚡ Combos rápidos (B + C)

| Combo | Tempo | Ação |
|-------|-------|------|
| **B + C** | tap rápido | Alterna **PRESET ↔ LIVE** |
| **B + C** | segurar **2 segundos** | Liga/desliga **TUNER** (afinador) |
| **B + C** | segurar **3 segundos** | Liga/desliga **Bluetooth** |

> Pisar em A/B/C/D **segurando** (no modo LIVE) = **salva** o preset no slot.

## 💡 Estados dos LEDs (decora!)

### Modo PRESET — LEDs dos knobs

| Estado | Significado |
|--------|-------------|
| Aceso fixo | Efeito ativo no preset |
| Apagado | Efeito desligado no preset |

### Modo LIVE — LEDs dos knobs

| Estado | Significado |
|--------|-------------|
| 🌑 Apagado | Módulo desligado |
| 🟢 Aceso fixo | Ligado, knob = som real (WYSIWYG) |
| ✨ Piscando | Ligado, mas knob ≠ som real (vai sincronizar quando você mexer) |

### LEDs dos footswitches

| Modo | Apagado | Aceso | Piscando |
|------|---------|-------|----------|
| **PRESET** | preset não selecionado | preset selecionado | (não usado normalmente) |
| **LIVE** | módulo desligado | (não usado) | módulo ligado |
| **Após salvar** | — | — | piscando rápido = salvo ✅ |

### LED indicador BT

| Estado | Significado |
|--------|-------------|
| 🌑 Apagado | BT desligado |
| ✨ Piscando | BT ligado, sem conexão |
| 🟢 Fixo | BT conectado |

### LED indicador de carga (traseira)

| Estado | Significado |
|--------|-------------|
| 🔴 Vermelho | Carregando |
| 🔵 Azul | Carga completa |
| 🌑 Apagado | Operando em bateria |

### Display

| Mostra | Significado |
|--------|-------------|
| Número 1–9 | Banco atual |
| Letra/nome de nota (`C`, `D`, `E`, etc.) | Modo TUNER |
| Letra com pontinho (`C` + ponto) | Sustenido (ex.: C#) |
| **L** piscando a cada 5s | Bateria fraca — **carregue!** |

## 🎚️ Knobs com 3 zonas (seleção de tipo)

Esses knobs têm 3 zonas físicas, cada uma = 1 tipo de efeito. Entre zonas = **desligado**.

| Knob | 1ª zona | 2ª zona | 3ª zona |
|------|---------|---------|---------|
| **MOD FX** | CHORUS (Mix) | PHASER (Depth) | TREMOLO (Mix) |
| **DLY MIX** | ANALOG | TAPE | DUAL |
| **RVB DECAY** | ROOM | SPRING | CLOUD |

| Knob | 9 zonas |
|------|---------|
| **TYPE** (AMP) | 9 modelos de pré-amplificador |
| **IR CAB** | Slot 1 = OFF, slots 2–9 = IRs carregados |

## 📋 Fluxos resumidos

### Carregar preset
```
BANK- / BANK+  →  display pisca número  →  A / B / C / D
```

### Editar e salvar no slot atual
```
Tap B+C (LIVE)  →  girar knobs  →  segurar A/B/C/D (atual)  →  piscou = salvo
```

### Salvar em outro slot
```
Tap B+C (LIVE)  →  girar knobs  →  BANK-/+ (escolhe banco)
                →  segurar A/B/C/D (destino)  →  piscou = salvo
```

### Afinar
```
Segurar B+C por 2s  →  toca corda  →  olha display + luzes  →  Tap B+C para sair
```

### Ativar Bluetooth + parear no celular
```
Segurar B+C por 3s  →  BT pisca  →  abrir M-EFCS app  →  Connect  →  BT fixo
```

## 🆘 SOS — algo deu errado

| Problema | O que tentar primeiro |
|----------|----------------------|
| Pedal **não liga** | Carregar 30 min com 5V/2A → tentar de novo |
| **Sem som no fone** | MASTER no mínimo? Cabo bom? Bateria? |
| Som **muito baixo** | MASTER, VOLUME (do AMP) e MIX (RVB/DLY) — checar todos |
| Som **distorcido demais** | GAIN alto + IR alto + MASTER alto = clipping. Reduz um. |
| **Não aparece no app mobile** | Localização ligada? BT ligado (3s)? |
| **Não detecta no PC** | Cabo USB de **dados** (não só carga)? |
| Bateria **L piscando** | Conecte USB-C com 5V/2A imediatamente |
| Knob "não faz nada" | Você está em modo **PRESET**? Knobs só agem no **LIVE**. |
| Preset trocou sozinho | Pisou em BANK-/+ sem perceber → confirmar com A/B/C/D |
